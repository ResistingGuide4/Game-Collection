Intro to Truth Tables

Before we get started talking about the logic of the gates, I have to introduce you to something. This is a truth table. It shows you the output of a circuit depending on the inputs. This can be used for logic gates as well as full circuits. In this program, I will be using abbreviations like I1, I2, and O1 to indicate the inputs and outputs. I also added colors to make them easier to see. This truth table explains the logic of the AND gate. As you can see, It only outputs ON if both inputs are ON.
